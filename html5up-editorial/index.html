HTTP Status Codes

1xx Informational
Code Name Description
100 Continue
101 Switching Protocol The server will switch the HTTP protocol used.
2xx Success
Code Name Description
200 OK Request Successful
201 Created Cia PUT
202 Accepted Recieved information but havent acted on it
203 Non-Authorative Information Resource Came from somewhere else.
204 No Content Successful but will not sent anything
205 Reset Content Reset the state of te resource
206 Partial Content A portion of the resource
3xx Redirection
Code Name Description
300 Multiple Choice Specify which confiturations
301 Moved Permanently Outdated URL
302 Found URL rewriting
303 See Other URL Rewriting
304 Not Modified Same ever since user loaded the resource
305 Use Proxy
306 -UNUSED-
307 Temporary Redirect Fetched the resource somewhere else
4xx Client Error
Code Name Description
400 Bad Request The Request was bad
401 Unauthorized
402 Payment Required For e-commerce use
403 Forbidden Not allowed to access the resource
404 Not Found The resource is not found
405 Method Not Allowed The request method used was not allowed by the server
406 Not Acceptable Cannot serve content
407 Proxy Authentication Required The proxy server requires authentication
408 Request Timed out The server took too long to respond to the request
409 Conflict The request is currently conflicting with the state of the server
410 Gone The resource doesnt exist anymore
411 Length Required
412 Precondition Failed
413 Request Entity Too Large The payload cannot be handled
414 Request URI Too Large The URI length was too large
415 Unsupported Media Type The server only accepts certain media types
416 Requested Range Not Satisfiable
417 Expectation Failed
426 Upgrade Required The HTTP protocol needs to be upgraded.
5xx Server Error
Code Name Description
500 Internal Server Error Misconfiguration of the server
501 Not Implemented The request method is not yet implemented in the server.
502 Bad Gateway Misconfigured Gateway
503 Service Unavailable
504 Gateway time-out The gateway is busy
505 HTTP Version Not Supported The HTTP version used by the client is not supported by the server
